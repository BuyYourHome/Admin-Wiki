# Worksheet Mode Rules

Use this folder to keep project-management spreadsheet redesign and rollout rules separated by worksheet mode.

Before doing workbook work, identify the active worksheet mode and read that mode's rule file. After completing work in a mode, write any reusable lesson to that same mode file before marking the work complete. Cross-cutting connector, rollback, and general rollout rules may remain in `Project Spreadsheet Expense Placement Rules.md`, but worksheet-specific rules should live here.

- `Profit Mode Rules.md`
