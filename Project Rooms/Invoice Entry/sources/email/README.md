# Routed Email Sources

Use this folder for contractor/vendor invoice emails routed to Invoice Entry by Email Monitor or OfficeAssist under Create Vendor Invoice Mode.

Each routed email source should preserve:

- sender,
- recipients,
- sent or received time,
- subject,
- Outlook message id or web link when available,
- body text,
- saved attachment paths or attachment-access blockers,
- the direct handoff summary that sent the item to Invoice Entry.

Invoice Entry uses these routed email sources and attachments to create structured invoice packets, perform duplicate checks, and preserve traceability back to the email handoff.
