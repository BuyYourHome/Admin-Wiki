# Geico Insurance Claim Project Room

## Purpose

Organize source documents, correspondence, notes, and review-ready outputs for the Geico insurance claim.

## Scope

Use this room to collect and evaluate claim-related materials, including policy documents, claim numbers, adjuster communications, estimates, photos, invoices, repair records, payment records, and timeline notes.

## Owner Intent

Prepare a clear working file so claim facts, open questions, deadlines, and next actions can be tracked without mixing unsupported assumptions into final communications or deliverables.
