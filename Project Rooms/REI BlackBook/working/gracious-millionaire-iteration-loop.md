# Gracious Millionaire Website Iteration Loop

Purpose: keep improving the REI BlackBook `Gracious Millionaire` website through short implementation passes, each followed by a practical suggestions list.

## Loop

1. Inspect the live site and REI BlackBook page manager for current pages, visible text, links, forms, and SEO fields.
2. Compare visible site text against Wes's current direction:
   - describe the book project,
   - do not post manuscript/chapter content yet,
   - use only descriptive copy, short quotes, or pictures that describe the book,
   - keep chapter links pointed to a shared editing-status page until chapter content is approved.
3. Make a small, reversible set of live edits in REI BlackBook.
4. QA the changed pages:
   - page loads,
   - visible text is book-focused,
   - links go where expected,
   - no manuscript content is exposed.
5. Record the implementation notes and a suggestions list in `working\gracious-millionaire-iteration-###.md`.
6. Start the next pass from the suggestions list Wes approves or prioritizes.
