## Source Archive

- Teams `Buy Your Home - Marketing\LD Evans\Source Materials\Video\Leaving Home\video\` contains the four downloaded MP4 files.
- Teams `Buy Your Home - Marketing\LD Evans\Source Materials\Video\Leaving Home\audio\` contains one M4A audio-only copy per recording for narration and voice analysis.
- Local `video\` contains one canonical English WebVTT caption file per recording.
- Large media and raw generated metadata are intentionally excluded from Git history by this folder's `.gitignore`.
- This README and the canonical WebVTT caption files are the durable, source-controlled index and text-access layer.

## Source Authority

- Treat the video and audio recordings as primary oral-history sources supplied by Wes.
- Treat the YouTube automatic captions as derivative working aids. Verify quotations, names, dates, and ambiguous wording against the audio before using them in book text.
- The videos support L.D. Evans book development, but factual claims should be corroborated with documents or additional testimony when practical.
- The recordings preserve the narrator's cadence, phrasing, emphasis, and storytelling approach for later narrative-voice study. Confirm narrator identity before making a formal attribution in a deliverable.
