# 4121 Tensity CMA Prototype

## Purpose

Create a repeatable project-room workflow for preparing a comparative market analysis (CMA) for a single-family property, using 4121 Tensity Dr, Raleigh, NC 27604 as the prototype property.

## Intended Output

- A review-ready CMA memo or packet for 4121 Tensity Dr.
- A reusable source inventory and comp-selection pattern for future CMA rooms.
- Clear separation between supported facts, public-data assumptions, and items requiring MLS, agent, or owner confirmation.

## Scope

This prototype uses public web sources only. It is not an appraisal and should not be treated as a final market opinion until MLS data, property condition, recent interior photos, seller goals, and any Buy Your Home internal property file are reviewed.

## Current Status

Prototype room created. Initial public source notes and a draft public-data CMA memo are in place.

## Folder Map

- `sources/` - source notes and links for subject property and comparable properties.
- `working/` - source inventory, conflict log, missing context, and valuation calculations.
- `outputs/` - review-ready CMA drafts.
