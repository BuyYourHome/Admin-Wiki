# Missing Context

## Required Before Final CMA

- MLS closed-sale export for the most comparable 3 bed / 2.5 bath Raleigh 27604 homes near Tensity Dr.
- MLS active and pending listings competing with the subject.
- Wake County property record for 4121 Tensity Dr.
- Current subject interior/exterior photos.
- Property condition and repair list.
- Occupancy status.
- Any HOA restrictions, dues, or neighborhood limitations.
- CMA purpose: acquisition, resale listing, refinance/lender support, estate planning, or internal project underwriting.
- Desired valuation premise: as-is, cleaned-up retail, after-repair value, quick-sale price, or conservative lender value.

## Useful But Not Required

- Prior appraisal or BPO.
- Insurance photos.
- Contractor scope or rehab budget.
- Rent estimate if the CMA will feed investment underwriting.
- Neighborhood-level DOM, list-to-sale ratio, and concession trends from MLS.

## Prototype Assumptions

- Public-data prototype assumes subject is broadly comparable to nearby Tensity homes built in 2002-2003.
- Public-data prototype does not assume the subject is renovated.
- Public-data prototype uses the public Redfin estimate, Realtor.com estimate signals, and public sale/listing data only.
