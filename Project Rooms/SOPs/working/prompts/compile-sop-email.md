# Compile SOP Email Prompt

Use this prompt after adding an SOP email to `Project Rooms/SOPs/sources/emails/`.

```text
Read the new SOP email source. Create or update the matching SOP page in `Project Rooms/SOPs/outputs/SOPs/` using `Project Rooms/SOPs/working/templates/sop-page.md`.

Preserve the source file unchanged. Link the SOP page back to the source email. If the email is unclear or conflicts with another source, add the issue to `Project Rooms/SOPs/working/Questions for Review.md`.

Also update `Project Rooms/SOPs/outputs/SOP Index.md` and `Project Rooms/SOPs/working/Source Map.md`.
```
