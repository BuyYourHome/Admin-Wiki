## Purpose

This Project Room is the canonical workspace for Buy Your Home SOP source material, SOP drafts, SOP review questions, and SOP maintenance outputs.

Use this room when creating, reconciling, reviewing, or updating SOPs from task instruction emails, the SOP spreadsheet, or operating-rule updates.

## Scope

In scope:

- SOP source emails.
- Extracted source notes.
- The Standard Operating Procedures workbook.
- Clean SOP Markdown pages.
- SOP index and source map.
- Open SOP review questions and duplicate/conflict notes.

Out of scope:

- Non-SOP admin wiki rules that already live at the vault root, such as `AGENTS.md`, `Document Scanning SOP.md`, and connector rules, unless they are being reconciled with this SOP corpus.
- Final Teams deliverables unless Wes explicitly asks to copy a finished file to Teams.
