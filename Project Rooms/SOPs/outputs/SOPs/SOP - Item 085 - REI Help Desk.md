# SOP - Item 085: Create REI Help Desk Ticket

## Purpose

Submit a support request to REI Blackbook when a system issue needs help desk support.

## When To Use

Use this when REI Blackbook has a technical problem, account issue, workflow issue, or other support need that cannot be resolved internally.

## Owner

- Role: Technical / Office Assistant
- Backup: Wes

## Category

Technical

## Inputs Needed

- Description of the issue
- Steps that caused the issue
- Screenshot or example contact/property, if available
- Contact email for replies

## Tools / Systems

- REI Blackbook
- REI Help Desk / Contact Us widget

## Steps

1. Open REI Blackbook.
2. From the main page, click Contact Us in the lower-right corner.
3. In the help box, briefly describe the issue.
4. Enter the contact email address for replies.
5. In the `How can we help you?` box, write the full issue description.
6. Include relevant details such as affected contact, property, workflow, browser, and error message.
7. Attach or reference screenshots if available.
8. Send the request.
9. Watch for the REI-generated reply.
10. Save or forward the support response if it affects an SOP or system setting.

## Decision Rules

- Include enough detail for REI to reproduce the issue.
- Do not send sensitive passwords in the ticket body.
- Escalate urgent business blockers to Wes after submitting the ticket.

## Common Mistakes

- Submitting a ticket with only a vague description.
- Forgetting to include the contact email.
- Not tracking the reply.
