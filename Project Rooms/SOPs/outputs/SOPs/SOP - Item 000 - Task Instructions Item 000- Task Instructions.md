# SOP - Item 000: Task Instructions Master List

## Purpose

Maintain a master reference list of Buy Your Home task instructions, task numbers, process groups, and assigned roles.

## When To Use

Use this page when reconciling the SOP spreadsheet against individual task instruction emails or when deciding which process page should be updated.

## Owner

- Role: Office Assistant / Operations
- Backup: Wes

## Category

Administration

## Inputs Needed

- `Project Rooms/SOPs/sources/spreadsheets/Standard Operating Procedures.xlsx`
- Task instruction emails in `Project Rooms/SOPs/sources/emails/`
- Current task assignment list in Teams, if available

## Tools / Systems

- Microsoft Teams
- Excel
- SOP wiki

## Steps

1. Open the SOP spreadsheet and review the task list by group and item number.
2. Compare each task number against the copied source emails.
3. Confirm whether each spreadsheet row has a matching source email.
4. For each matched email, maintain a draft SOP page in `Project Rooms/SOPs/outputs/SOPs/`.
5. For missing emails, leave the spreadsheet row in `Project Rooms/SOPs/outputs/SOP Index.md` as `Needs source email`.
6. For moved, replaced, or discontinued tasks, keep the old task page but mark it as superseded and point to the active task.
7. Update `Project Rooms/SOPs/working/Questions for Review.md` when task ownership, status, or source history is unclear.

## Decision Rules

- Do not delete historical task numbers just because they were moved or replaced.
- Use the spreadsheet as the master inventory, but use the most recent email when email and spreadsheet titles conflict.
- Keep raw source files unchanged.

## Exceptions

- Item 000 is not an operating procedure. It is an inventory/control page.
