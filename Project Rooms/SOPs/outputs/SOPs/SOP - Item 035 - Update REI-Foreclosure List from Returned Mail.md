# SOP - Item 035: Update REI and ForeclosureList From Returned Mail

## Purpose

Record returned mail in REI and the ForeclosureList so returned addresses are not mailed again.

## When To Use

Use this whenever returned mail is received.

## Owner

- Role: Marketing / Office Assistant
- Backup: Wes

## Category

Marketing

## Inputs Needed

- Returned envelope
- Contact name or address
- Zone information, if known
- Access to REI and ForeclosureList

## Tools / Systems

- REI Blackbook
- ForeclosureList workbook
- Physical returned mail envelope

## Steps

1. Process returned mail as it comes in.
2. Search for the contact in REI by name.
3. Open the contact profile.
4. Go to Tags.
5. Add the `Returned` tag.
6. Write `marked returned` on the envelope.
7. Mark the zone on the envelope if available.
8. Open the ForeclosureList.
9. Find the matching record.
10. Set the Mail column to `Don't Mail`.
11. Continue with the current follow-up process in Item 90.
12. If the returned mail indicates a potentially valuable property lead, continue to the review/binder workflow before discarding the envelope.

## Decision Rules

- Returned mail should stop future mail to that address unless Wes directs otherwise.
- Complete returned mail processing before moving to the potential property visit binder.

## Common Mistakes

- Tagging the contact in REI but forgetting the ForeclosureList update.
- Throwing away the envelope before zone or review handling is complete.
