# Estate Documents

## Purpose

Create a dedicated Project Room for Wes's estate documents so source files, working notes, and answers to future estate-related questions are kept together and reviewed from a clean source inventory.

## Intended Output

- Organized estate-document source set.
- Source inventory showing which documents are authoritative, background, duplicate, outdated, unclear, or missing.
- Question-and-answer notes that cite the source documents used.
- Review-ready summaries or action lists in `outputs\` when requested.

## Current Scope

Included:

- Estate planning documents provided by Wes.
- Source notes or summaries created from estate-related emails, scans, PDFs, Word documents, spreadsheets, or other records.
- Questions and answers about the estate documents, only when supported by inventoried sources.

Excluded unless Wes specifically adds them:

- Final legal advice.
- Unsourced assumptions about estate plans, beneficiary designations, asset ownership, tax treatment, probate, or trust administration.
- Teams copies or external deliveries.

## Source Locations

- Room sources: `sources\`
- Working notes: `working\`
- Outputs: `outputs\`

## Source Inventory Status

- Authoritative sources: pending
- Duplicate/outdated sources: pending
- Missing context: pending

## Answering Rules

- Answer estate-document questions from authoritative sources whenever possible.
- Cite the specific source file or source note used for every factual answer.
- Mark unsupported items as `[UNSUPPORTED]` instead of filling gaps.
- Flag legal, tax, beneficiary, ownership, or probate questions that need attorney or CPA review.
- Preserve original source files. Edit only working notes or output drafts.

## Decisions Needed

- Which estate documents should be copied into `sources\`.
- Whether Wes wants this room to include related non-estate financial records, such as account statements, insurance records, beneficiary confirmations, or business ownership records.
- Whether any outputs should be copied outside the wiki after review.
