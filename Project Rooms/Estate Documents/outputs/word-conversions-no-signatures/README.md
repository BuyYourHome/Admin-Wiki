# Word Conversions - Page Image Version

This folder contains Word documents generated from the scanned estate PDFs in `sources\Office Admin Estate`.

Conversion limits:

- The source PDFs are image-only scans, not attorney-delivered searchable PDFs.
- These Word files embed rendered page images from the source PDFs.
- Signatures, initials, notary stamps, lines, binder holes, scan streaks, and other source artifacts are intentionally preserved.
- The text is not cleanly editable because each page is an image inside Word.
- Use these as visual reference copies, not certified legal originals.
- `2156 Haig Point Deed.pdf` and `Living Will - Jenny Hollinger.pdf` rendered as essentially blank scanned pages, so their Word files also appear blank.
