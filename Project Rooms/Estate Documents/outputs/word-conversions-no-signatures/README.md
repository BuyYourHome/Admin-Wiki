# Word Conversions - No Signatures

This folder contains OCR-based Word conversions of the scanned estate PDFs in `sources\Office Admin Estate`.

Conversion limits:

- The source PDFs are image-only scans, not attorney-delivered searchable PDFs.
- These Word files are OCR drafts for reference and editing, not certified legal originals.
- No scanned images were embedded in the Word files, so signature images and notary stamps were not carried over.
- Obvious signature/initial/notary areas are represented with text and blank lines when OCR detected related labels.
- `2156 Haig Point Deed.pdf` and `Living Will - Jenny Hollinger.pdf` rendered as essentially blank scanned pages, so their Word files state that no readable text was detected.
