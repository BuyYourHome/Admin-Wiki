## Core Thesis

The video introduces the ACE folder framework as a way to organize a personal knowledge system around three recurring thinking modes: knowledge, time, and action. ACE maps those modes to `Atlas`, `Calendar`, and `Efforts`, giving both humans and AI a stable orientation system for deciding where information belongs.

## Implementation-Relevant Concepts

- Organize knowledge systems around durable thinking modes, not just arbitrary folders.
- Use `Atlas` for knowledge and reference material.
- Use `Calendar` for time-based notes, logs, meeting notes, and reflections tied to dates.
- Use `Efforts` for projects, outcomes, and action-oriented work.
- Start simple; add structure only after repeated use proves it is needed.
- Support both folder navigation and linked maps/home notes.
- Let AIOS maps explain what belongs in each headspace so tools can place and find information consistently.
