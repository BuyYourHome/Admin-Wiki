# Gracious Millionaire Website Logo Handoff

## Asset

- `gracious-millionaire-logo-concept-2026-07-09.png`

## Website Use

Use this image as the Gracious Millionaire logo on the website.

Recommended placement:

- Site header or navigation brand area.
- Footer brand area.
- Book landing page hero or intro section where the Gracious Millionaire identity is introduced.

Recommended handling:

- Keep the logo visually restrained and literary; do not pair it with money, luxury, or flashy wealth imagery.
- Use it against a plain light, cream, or very dark charcoal background with enough contrast.
- If the site needs a smaller mark, crop/export the standalone `GM` monogram from this source rather than shrinking the full horizontal lockup too far.
- Add descriptive alt text: `Gracious Millionaire logo`.

Design intent:

- The mark should support the book's faith-centered memoir theme: stewardship, trust, humility, marriage, favor, and lessons learned through real estate.
- Avoid using it as a prosperity or luxury brand mark. The Gracious Millionaire project-room guidance treats real-estate wealth as the context, not the subject.
