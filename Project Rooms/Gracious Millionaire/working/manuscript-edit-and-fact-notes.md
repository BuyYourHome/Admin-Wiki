# Manuscript Edit And Fact Notes

## Purpose

Use this file to capture quick corrections, factual updates, and changes in how Wes wants the manuscript to frame a person, event, theme, or source.

This file is not manuscript prose. Treat each note as editorial direction for the next relevant revision pass.

## How To Use

Add new notes under `Open Notes` with:

- Date added.
- Affected section or chapter, if known.
- Type: `fact`, `framing`, `tone`, `structure`, `source`, or `question`.
- Direction from Wes.
- Status: `open`, `applied`, `deferred`, or `needs clarification`.

When a note is applied to the manuscript, move or copy it into `Applied Notes` and identify the manuscript version or file where it was handled.

## Drafting Rule

These notes control over prior drafts, provisional profiles, and older style guidance where they conflict. Do not treat Wes's quick note wording as final prose unless he explicitly says to use the exact words.

For framing notes, preserve the intended perception without overstating facts beyond the source material.
