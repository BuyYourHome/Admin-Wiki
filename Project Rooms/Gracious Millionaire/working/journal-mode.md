# Journal Mode

Use Journal mode when Wes asks for a Gracious Millionaire journal conversation, asks to journal, or says the conversation is not ready to become book source material.

Journal mode is an interactive conversation mode for thinking, reflection, questions, and private development around Gracious Millionaire. It is not a manuscript mode and is not source material unless Wes explicitly asks to compile or promote a journal conversation for source use.

## Operating Rules

- Always identify the response as `Journal mode` while working in this mode.
- Keep Journal mode conversation separate from manuscript modes, routed source intake, and publication workflows.
- Do not include Journal mode content in Quick, Rewrite, Dialogue, Interview, or any other manuscript output unless Wes explicitly asks for that specific journal material to be compiled for manuscript/source use.
- Do not add Journal mode content to `working/source-inventory.md`, `working/book-architecture.md`, manuscript chapter files, output manifests, or review packets unless Wes explicitly asks to convert it into source material.
- Do not email Journal mode activity to Jenny, reviewers, or any outside recipient. If Wes asks for an email copy of Journal material, send it only to Wes unless he gives a different explicit instruction and the Admin email rules allow it.
- Journal conversations may remain segregated from other Journal conversations. When a journal conversation is saved locally, use a separate file under `working/journal/` with a date/topic filename.
- A saved Journal file is private working material, not authoritative source material. Mark it `journal/private-working` until Wes explicitly promotes it.
- If Wes asks to compile a Journal conversation for possible book use, create a separate compiled source candidate under `working/journal/compiled/` first. Do not add it to source inventory or manuscript materials until Wes approves that promotion.

## Promotion To Source

Journal mode material becomes book source material only after Wes explicitly asks for it. When that happens:

1. Compile the relevant journal conversation into a source-candidate file.
2. Preserve the original journal conversation separately.
3. Ask or confirm whether the compiled candidate should become source material if the instruction is ambiguous.
4. After explicit approval, add the compiled candidate to the source inventory with status `journal-promoted/source`.
5. Only then may manuscript modes use the promoted material.
