# Admin Wiki Maintenance

## Purpose

Use this room for Admin wiki maintenance work such as branch cleanup notes, repository hygiene reviews, rule consolidation, and cross-project operating-system maintenance that does not belong to a more specific Project Room.

## Scope

- Keep maintenance reports and cleanup plans under this room.
- Do not use this room as a substitute for a specialized Project Room when one exists.
- Leave unrelated dirty files and project-specific work out of maintenance commits.
