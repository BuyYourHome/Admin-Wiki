# SOP - Item 112: File Year-End W-2s

## Purpose

File year-end W-2/W-3 forms through Social Security Business Services Online and print employee copies.

## When To Use

Use this after year-end payroll records are complete and W-2s need to be filed.

## Owner

- Role: Accounting / Wes
- Backup: Tax professional or approved accounting support

## Category

Accounting / Payroll Tax

## Inputs Needed

- Payroll records for the year
- Employee wage and tax withholding totals
- Federal and state wage/tax information
- SSA Business Services Online access
- Login.gov access and one-time code recipient

## Tools / Systems

- SSA Business Services Online
- Login.gov
- Payroll records
- NC eNC3, if North Carolina state tax was withheld

## Steps

1. Go to SSA Business Services Online.
2. Sign in through Login.gov using approved credentials.
3. Select Report Wages to Social Security.
4. Choose File W-2/W-3 Online.
5. Select Create Forms W-2/W-3 Online for manual entry.
6. When asked whether filing is for self-employed income, answer No for S-Corp employee wages.
7. Select Employer as the kind of payer.
8. Select the correct employer type, usually None Apply unless the tax advisor says otherwise.
9. Select 941 as the filing type when filing regular payroll wages.
10. Create W-2s using payroll records.
11. Enter wages in Box 1 from salary only, not S-Corp distributions.
12. Enter federal tax withheld in Box 2.
13. Enter Social Security and Medicare wages in Boxes 3 and 5.
14. Enter Social Security and Medicare taxes in Boxes 4 and 6.
15. Leave Box d blank unless a control number is used internally.
16. Complete Box 12 only if applicable, such as company-paid health insurance or retirement items.
17. Complete North Carolina state boxes 15 through 17 if NC taxes were withheld.
18. Run validation in BSO.
19. Fix any validation errors.
20. Submit the filing.
21. Save the confirmation number.
22. Print W-2 copies for employees and company records.
23. File state wage reporting through NC eNC3 if required.

## Decision Rules

- W-2 wages include salary, not S-Corp distributions.
- W-2 totals should match the four quarterly Form 941 filings.
- Do not fix prior-quarter errors by changing later filings; amend the prior filing when needed.
- Do not store government login passwords or one-time codes in this SOP.

## Common Mistakes

- Including S-Corp distributions as W-2 wages.
- Leaving W-2 totals inconsistent with Form 941 totals.
- Forgetting state wage filing when NC tax was withheld.
