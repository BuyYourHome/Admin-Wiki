# SOP - Item 066: Add Property Insurance

## Purpose

Request property insurance coverage for a new property and make sure interested parties are included.

## When To Use

Use this when a property enters rehab or ownership and needs to be added to the appropriate company insurance account.

## Owner

- Role: Property Rehab / Office Assistant
- Backup: Wes or Jenny

## Category

Property Rehab

## Inputs Needed

- Property address
- Company that owns the property: BYH or SYH
- Mortgage company name and account number, if applicable
- Private lender names and addresses, if applicable
- Property spreadsheet/contact information

## Tools / Systems

- Email
- Property spreadsheet
- Insurance folder in inbox

## Steps

1. Open the company inbox.
2. Choose the correct company that owns the property.
3. Open the Insurance folder for reference.
4. Start a new email to the insurance contact.
5. Ask for the property to be added to the appropriate BYH or SYH insurance account.
6. Include the full property address.
7. Add interested parties from the property spreadsheet or contact records.
8. Include first mortgage information when applicable, including company name and account number.
9. Include private money lender names and mailing addresses when applicable.
10. If interested party information is missing from the property spreadsheet, add it or flag it for completion.
11. Send the request.
12. Save or file any confirmation response with the property records.

## Decision Rules

- Use the owning company account, not the person requesting the insurance.
- Add mortgage companies and private lenders as interested parties when required.
- Do not send incomplete lender information unless Wes approves.

## Common Mistakes

- Requesting coverage under the wrong company.
- Forgetting lender or mortgage interested-party information.
- Not saving confirmation.
