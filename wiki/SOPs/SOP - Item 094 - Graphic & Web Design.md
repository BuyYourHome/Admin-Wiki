# SOP - Item 094: Graphic and Web Design Reference

## Purpose

Maintain brand style references for Buy Your Home-related web and graphic design work.

## When To Use

Use this when creating or updating web pages, landing pages, graphics, or marketing materials for the listed brands/sites.

- Approved copy and image requirements

## Steps

1. Confirm which website or brand the design is for.
2. Use the matching color and font references above.
3. Select photos from approved sources when original property or brand images are not available.
4. Keep design files or exports in the project folder associated with the design work.
5. Send drafts for review before publishing.
6. Update this page when a brand style changes.

## Decision Rules

- Use actual property or project images when available.
- Use stock images only when they fit the specific project and are legally usable.
- Treat this page as a reference, not a full design brief.
