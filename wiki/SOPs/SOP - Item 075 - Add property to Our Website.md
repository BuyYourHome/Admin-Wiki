# SOP - Item 075: Add Property to Website

## Purpose

Publish or update a property on the BuyYourHomeLLC.com website through REI Blackbook.

## When To Use

Use this when a property should appear on the website as available for private lender funds or when a sold property should be marked sold.

## Owner

- Role: Property Rehab / Marketing
- Backup: Wes

## Category

Property Rehab / Marketing

## Inputs Needed

- Property address
- Bed, bath, and square footage details
- Private lender funding amount, if applicable
- Property photo from Zillow or approved source
- Market status: Under Contract or Sold

## Tools / Systems

- REI Blackbook
- BuyYourHomeLLC.com property pipeline
- Zillow or approved image source

## Steps

1. Log in to REI Blackbook.
2. From Deals, open Property Pipeline.
3. Search for the property address.
4. Open the property record.
5. Click the Marketing tab.
6. Open Property Details.
7. Turn Website Status to Publish.
8. Turn Featured Website Status to Publish when the property should be featured.
9. In Property Basics, enter or verify beds, baths, and square footage.
10. Save the property basics.
11. At the top-right of the page, set Market Status to Under Contract when the property is available for private lender funds.
12. In Property Description, use the appropriate private-lender title and short description.
13. Save a property photo from Zillow or another approved source.
14. Upload the property photo under Property Images.
15. Save the property record.

## Sold Property Update

1. Open the property in REI Blackbook.
2. Follow the same path to the Marketing and Property Details area.
3. Change Market Status to Sold.
4. Update the title to `SOLD`.
5. Update the short description to indicate the property was another Buy Your Home success.
6. Save the property record.
7. Confirm the property no longer appears as available for investors but still displays as sold where intended.

## Decision Rules

- Use Under Contract when the property should appear on the private lenders page.
- Use Sold after the sale is complete.
- Only use approved property photos.

## Common Mistakes

- Publishing without beds, baths, square footage, or image.
- Leaving sold property as available for private lender funds.
- Uploading the wrong property photo.
