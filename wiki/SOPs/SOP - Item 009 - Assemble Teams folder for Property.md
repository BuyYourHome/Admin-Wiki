# SOP - Item 009: Assemble Teams Folder for Property

## Purpose

Create the working Teams folder and initial buying documents for a property that is moving forward.

## When To Use

Use this when a project is progressing as HOT and needs a formal property folder.

## Owner

- Role: Acquisition / Office Assistant
- Backup: Wes

## Category

Acquisition

## Inputs Needed

- Property address
- Company name: BYH or SYH
- Access to the All Properties Teams folder
- New Property Template Folder

## Tools / Systems

- Microsoft Teams
- All Properties folder
- New Property Template Folder

## Steps

1. Open Teams.
2. Go to the All Properties file area.
3. Locate the New Property Template Folder.
4. Right-click the template folder.
5. Choose Copy to and copy the folder to the same location.
6. If Teams asks whether to keep both copies, choose Yes.
7. Rename the copied folder using this format: `BYH - Property Address` or `SYH - Property Address`.
8. Open the renamed property folder.
9. Create a new folder named `BUYING`.
10. Move these documents into the `BUYING` folder:
    - Authorization to Release Information and Escrow Letter
    - Standard Purchase and Sales Agreement
    - Seller's Acknowledgements
    - Letter of Agreement and Addendum-CYA
11. Fill out the forms for Wes as completely as possible using available property information.
12. Flag any missing information for Wes before final use.

## Decision Rules

- Use BYH or SYH based on which company owns or will own the project.
- Do not modify the original template folder.
- If item 95 appears in old instructions, treat this item as the active Teams setup process.

## Common Mistakes

- Renaming the template instead of the copied folder.
- Forgetting to create the `BUYING` folder.
- Moving the wrong documents or leaving buying documents outside the `BUYING` folder.
