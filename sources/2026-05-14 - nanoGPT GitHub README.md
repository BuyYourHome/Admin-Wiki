## Raw Notes

The README describes nanoGPT as a compact repository for training or finetuning medium-sized GPT models. It says nanoGPT is a rewrite of minGPT and focuses on a plain, readable implementation. The README also notes a November 2025 update: nanoGPT has a newer cousin called nanochat, and nanoGPT is now old and deprecated but kept available for posterity.

## Key Claims

- nanoGPT is a simple repository for training or finetuning GPT-style models.
- It rewrites minGPT with more emphasis on practical training.
- Its core code is intentionally small and readable.
- As of the README's November 2025 update, nanoGPT is deprecated in favor of nanochat.
