# AIOS Efforts Index

This index groups active work and deliverable areas in the Admin wiki. It is an ACE overlay index, not a folder migration.

## Purpose

Use this when the request asks to:

- draft or revise a deliverable,
- analyze multiple sources,
- implement a workflow,
- update a project package,
- build a report,
- or manage active project work.

## Project Room Rule

Substantial multi-source work belongs under:

```text
C:\Codex\Wiki Files\Project Rooms
```

Follow `Project Room Workflow.md` before drafting from multiple sources.
