# AIOS Atlas Index

This index groups durable knowledge and reference areas in the Admin wiki. It is an ACE overlay index, not a folder migration.

## Purpose

Use this when the request asks:

- what the rule is,
- where a policy lives,
- how a workflow is supposed to operate,
- which source is authoritative,
- or how AI should navigate the wiki.
