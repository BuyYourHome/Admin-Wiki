# Capitalization Structure

## Capital Source Rule

The project is now modeled on the basis that all design, development, build, and delivery capital is raised from Class A and Class B members.

The model does not assume a development loan or construction loan. The only new loan contemplated in the workbook is a stabilization refinance/takeout loan after the project reaches the refinance point.

## Class A Beneficiary Structure

Class A participants are anticipated to participate as preferred capital beneficiaries of the trust structure.

### Class A Preferred Return

* 8% cumulative preferred annual return
* Accruing from date of contribution
* Calculated pro rata based on staged capital calls

Preferred returns are anticipated to accrue throughout:

* entitlement,
* development,
* construction,
* and stabilization.

Class A preferred return accrues while capital is outstanding, but the current waterfall pays member capital back before paying the accrued Class A preferred return. After Class A capital, Class B capital, and accrued Class A preferred return have been satisfied, Class A also participates in residual distributions through the Class A/Class B residual pool.

---

## Class A Distribution Priority

Under the current Capital Recovery First / Deferred Preferred Return waterfall, proceeds are anticipated to flow generally in the following order:

1. Pay existing first mortgage debt
2. Pay existing private debt principal and any then-due private debt interest obligations
3. Return Class A contributed capital
4. Return/pay Class B capital from remaining proceeds and operating/sale cash flow according to the Class B terms
5. Pay accrued Class A preferred return
6. Residual distributions according to the adjustable Sponsor/Class pool split

## Class B Beneficiary Structure

Class B participants are anticipated to participate as subordinate participating beneficiaries within the trust structure.

Class B capitalization is modeled as the completion-capital source required to fully fund the project after Class A capital. The working capitalization case assumes sponsor-controlled post-frame construction, including sponsor-owned equipment and sponsor-controlled crew capacity.

In the current workbook, Class B is calculated as:

* selected total project cost,
* less Class A capital.

# Annual Cash Roll-Forward

The Annual Model now includes a cash roll-forward instead of treating each year's cash as isolated.

The Annual Model shows:

* current-year cash before refinance/sale,
* net refinance or sale proceeds,
* cash available before distributions,
* waterfall distributions,
* ending cash balance,
* loan balance,
* and property/refinance value.

Cash available before distributions is calculated as:

```text
Prior-year ending cash balance
+ current-year cash before refinance/sale
+ net refinance/sale proceeds
- reserve funding
```

Waterfall distributions are then pulled from the Waterfall tab. Ending cash balance is calculated after those distributions. The dedicated repairs and maintenance reserve is funded before waterfall distributions; once the target is funded, remaining cash flows through the waterfall.

Under the current waterfall settings, available cash is distributed each year, so the modeled ending cash balance is currently **0** in the early operating years. The structure is now in place to carry retained cash forward if the waterfall or reserve assumptions later leave cash undistributed.

The workbook's Returns Table is embedded below the Waterfall table. It now shows Initial Capitalization followed by annual Year 1-Year 10 cash flows, then Total Profit, Total Distributions, Equity Multiple, ROI %, and IRR. The prior visible Year 0 label has been removed from the Returns Table so Year 1 is the first operating year shown. A Management row has been added and is calculated dynamically as **6.00% of each year's effective gross income (EGI)**.

The Returns Table now separates `Capital Investment` from `Basis / Property Payment`. For Class A and Class B, capital investment and basis are currently the same. For Sponsor, `Sponsor Capital Investment` is a separate editable assumption from `Sponsor Property Payment / Basis`. Sponsor ROI and equity multiple use Sponsor Capital Investment, while the property payment/basis remains tied to the Initial Capitalization sponsor property payment.

The Waterfall now carries unpaid Class A and Class B capital directly through the Class A and Class B capital balance columns. Separate ending-capital columns were removed; each year's capital balance equals the prior year's capital balance less the prior year's capital returned.

The Waterfall also carries unpaid Class A preferred return directly through `End of Year Pref Balance`. Preferred return accrues while Class A capital remains outstanding, but the current waterfall pays Class A capital and Class B capital before paying accrued Class A preferred return.

The Annual Model, Waterfall, and Returns Table now zero out years after the selected sale/exit year. The sale year itself may still show final operations, sale proceeds, loan payoff, and waterfall distributions; years after the sale year should show no further activity.

The refinance and sale/exit years are selected from option buttons beside the Waterfall table. Buttons are shown only for Years 1-10, with no button on the internal Year 0 / Initial Capitalization row. The refinance option buttons use `Waterfall!R3`, named `SelectedWaterfallRefiOption`, and the exit option buttons use `Waterfall!S3`, named `SelectedWaterfallExitOption`.

The refinance loan is modeled as the lesser of:

1. **70% of stabilized value**, or
2. **total existing debt plus Class A and Class B capitalization**.

Based on the current workbook values:

* modeled refinance loan amount: approximately **732,276**,
* net refinance proceeds after existing debt payoff: approximately **538,276**,
* indicative amount remaining after existing debt and Class A capital: approximately **238,276**.

The refinance loan is not modeled as a source of development or construction capital. It is modeled as a takeout loan after stabilization, with net proceeds flowing through the capital-first waterfall.
