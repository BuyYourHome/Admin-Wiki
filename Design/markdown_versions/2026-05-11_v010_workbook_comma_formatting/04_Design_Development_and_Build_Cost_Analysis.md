# Planning Cost Framework

This cost analysis separates the project into three major cost categories:

1. Design and entitlement
2. Development and pre-construction
3. Build and delivery

The project is assumed to include:

* approximately **5,000-6,000 SF** of residential building area,
* 8 private residential suites,
* shared common areas,
* post-frame construction,
* slab-on-grade construction,
* engineered onsite wastewater,
* private utility infrastructure,
* sponsor-owned equipment,
* sponsor-controlled crew capacity,
* and long-term rental operation rather than speculative resale.

This file keeps two cost lenses in view:

1. **Sponsor-controlled working case**: reflects post-frame construction, sponsor-owned equipment, and sponsor-controlled crew capacity.
2. **Outside-market benchmark**: reflects a more conservative third-party development and construction pricing scenario.

# Debt Payoff Constraint

The workbook now models project debt payoff at stabilization/refinance as the lesser of:

1. **70% of stabilized value**, or
2. **Class A plus Class B capitalization**.

Based on current workbook values:

| Item | Amount |
|---|---:|
| Stabilized value | ~1,064,090 |
| 70% of stabilized value | ~744,863 |
| Class A + Class B capitalization | 1,300,000 |
| **Modeled project debt payoff** | **~744,863** |

The current payoff cap is therefore driven by stabilized value rather than available Class A and Class B capitalization.

# Preliminary Feasibility Observations

## Most Important Cost Drivers

The largest cost drivers are likely to be:

* building size and construction cost per square foot,
* number of private bathrooms and kitchenettes,
* engineered wastewater system requirements,
* sitework and utility infrastructure,
* the extent to which sponsor-owned equipment and sponsor-controlled labor reduce third-party contractor pricing,
* financing terms,
* and contingency reserves.

## Relationship to Current Return Model

The workbook now models refinance, distributions, and investor returns with Class B sized as completion capital for the high sponsor-controlled cost case. The project debt payoff at refinance is constrained by the lesser of 70% stabilized value or total Class A and Class B capitalization.
