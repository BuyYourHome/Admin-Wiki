# Repository Location Rule

The working Git/wiki repository is:

`C:\Codex\Wiki Files`

Do not use the Teams-synced `Wiki Files` folder as an ongoing wiki repository or automatic publishing destination.

Teams folders should receive only specific final deliverables when explicitly requested, such as a finished SOP, spreadsheet, or one-off file.

Admin workflow skills are also source-controlled in this repository under `C:\Codex\Wiki Files\skills`. Local Codex skill folders are installed copies; see [[Codex Skill Source Rule]].
