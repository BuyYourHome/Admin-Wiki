# Compile SOP Email Prompt

Use this prompt after adding an SOP email to `sources/emails/`.

```text
Read the new SOP email source. Create or update the matching SOP page in `wiki/` using `templates/sop-page.md`.

Preserve the source file unchanged. Link the SOP page back to the source email. If the email is unclear or conflicts with another source, add the issue to `wiki/Questions for Review.md`.

Also update `wiki/SOP Index.md` and `wiki/Source Map.md`.
```
