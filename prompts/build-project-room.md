# Build AI Project Room

Use this prompt when starting or refreshing a Project Room.

```text
Build or refresh the AI Project Room for this task.

Do not draft the final deliverable yet.

Review the files and notes in the room's sources folder. Then create or update:

1. working/source-inventory.md
   - filename
   - source type
   - date or last modified date
   - one-line summary
   - status: authoritative, background, duplicate, outdated, unclear, missing
   - why this status was assigned

2. working/duplicate-and-conflict-log.md
   - overlapping sources
   - outdated versions
   - conflicting facts, numbers, or instructions
   - recommended source of truth
   - decisions needed from Wes

3. working/missing-context.md
   - facts or decisions needed before drafting
   - unsupported claims likely to matter
   - specific source, data, or human decision needed

After the room is prepared, summarize whether it is ready for drafting. If not ready, list the smallest set of missing inputs needed.
```
