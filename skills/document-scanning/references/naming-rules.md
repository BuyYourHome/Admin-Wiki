# Naming Rules

Default output filename:

`YY-MM-DD - Company.pdf`

Use the statement date, invoice date, or billing cycle closing date shown in the document.

Examples:

- `26-05-05 - First Citizens Bank VISA.pdf`
- `26-05-02 - Lowe's Pro.pdf`
- `26-04-30 - SECU Mortgage.pdf`

Do not use the scan date unless no document date is visible. If using scan date, state that in the log.

When a destination file already exists, append a numeric suffix:

- `26-05-05 - First Citizens Bank VISA (2).pdf`
- `26-05-05 - First Citizens Bank VISA (3).pdf`
